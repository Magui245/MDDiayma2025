# Guide de Déploiement - Boutique Diayma 2025

## Prérequis
- Windows 10 ou 11
- .NET Core Runtime 2.0 ou supérieur

## Installation
1. Téléchargez le fichier ZIP
2. Extrayez tout le contenu dans un dossier
3. Double-cliquez sur `RUN_APPLICATION.bat`

## Dépannage
Si l'application ne démarre pas, assurez-vous d'avoir installé le Runtime .NET Core.
